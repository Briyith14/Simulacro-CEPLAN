# Análisis de Indicadores CEPLAN

## Descripción
Base de datos limpia con indicadores de desarrollo de 195 provincias peruanas.

## Variables incluidas
- Provincia
- Porcentaje de nacidos con bajo peso
- Porcentaje de desnutrición crónica (menores 5 años)
- Porcentaje de anemia (6-35 meses)
- Índice de Vulnerabilidad a la Inseguridad Alimentaria (IVIA)
- Índice de Densidad del Estado (IDE)
- Índice de Desarrollo Humano (IDH)
- Devengado per cápita - Actividad
- Devengado per cápita - Inversión

## Fuente
Data original: CEPLAN

## Estructura
- 195 observaciones (provincias)
- 9 variables
